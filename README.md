# 1차 프로젝트 - 전통주 경매 쇼핑몰

중간 프로젝트 - 전통주 경매 쇼핑몰

개요

- 신제품 홍보를 위한 경매 기능, 구매자들 간 정보공유 목적의 게시판 기능을 추가한 쇼핑몰 구현

주요 기능

1. 로그인 / 회원가입 기능

2. 경매 관리 기능

3. 상품 관리 기능

4. 게시판 관리 기능

5. 경매 기능

6. 주문, 결제 기능

7. 이미지 관리 기능

구성원

팀장 

- 김다훈 ( 경매 관리, 상품 관리, 로그인 기능 )

팀원 

- 김도현 ( 회원가입 기능, UI )

- 유다한 ( 상품 관리, 결제 기능 )

- 이찬희 ( 게시판 관리, 이미지 관리 )

- 위태욱 ( 게시판 관리, 주문 기능 )

- 지대한 ( 주문 기능, 리스트 )
